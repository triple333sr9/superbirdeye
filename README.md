# SuperBirdEye
Developer: Jordan Yu @triple333sR9
### Intro
This repository was used to train a model that when given an image of a bird, can output the keypoints of the bird's eyes and beak where visible.

### Environment
##### Data set : CUB-200-2011
Citation:

##### Structure: Pytorch
##### Programing language: Python 3.13.9

##### Pretrained Models Used
- Resnet
- Densenet
- YOLOv8n-pose
- SuperAnimal-Bird
Other models attempted but removed due to poor performance include CNN without backbone, efficientnet, autoencoder and VAE.

##### Hyperparameters selected
- resnet50
- wing loss function
- 5e-05 learning rate
- drop out rate of 0.2
- heavy augmentation (see configuration for details)
- plateau scheduler
See configuration for more complete list of hyperparameters tested.

### Run Code
##### Run Training
- main.py allows specifying the training of 1 model and uses the train.py template to train
- to run a grid search use either grid_search.py or main.py without setting any args, this will go through the training process for all model and configuration options (as specified and generated through config.py)

##### Evaluation
- To evaluate a single model use evaluate.py (for yolo use evaluate_yolo.py)
- To evaluate the results of the grid search use analyze_grid_search.py (evaluate_grid.py generates the JSON report files which occurs during the training process)

##### Model Usage
To generate usable images for interference, use detect_and_keypoint.py (yolov11) to identify images of birds and crop it to a size more similar to training data.
Run test_on_real.py to use the model. It is set to test resnet, densenet and yolov8, adaptions may be required to run for just one model.
